### Instalation

##### Abra a pasta:
`$ cd MyTrainer_Cordova/`

##### Adicione a plataforma e construa o app:

`$ cordova build ios`

##### Inicie o app (dispositivo ou emulador):
Android:
`$ cordova run android`

iOS:
`$ cordova run ios`
